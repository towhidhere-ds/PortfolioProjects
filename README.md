# Data Analytics Portfolio Repository

This repository contains a collection of my practical data analytics projects using SQL, Python, Power BI, Excel, and Tableau. Each project reflects a real problem-solving scenario where I work through data cleaning, exploration, visualization, and insight generation.

---

## Included Projects

### 1. HR Analytics Dashboard (Power BI)

A dashboard that helps HR decision-makers understand attrition patterns, job satisfaction scores, age group trends, and gender-based turnover.

**Skills used:** Power BI modeling, DAX measures, KPI visuals, slicers, storytelling with data

---

### 2. Amazon Price Tracker - Python Web Scraper

A Python script that scrapes Amazon product prices and emails a notification if the price drops below a target amount.

**Skills used:** BeautifulSoup, requests, SMTP email automation, scheduling logic

---

### 3. COVID-19 SQL Exploration

Dataset-based SQL project analyzing deaths, confirmed cases, and vaccination progress across countries. It answers questions about mortality trends, population-level spread, and timeline behavior.

**Skills used:** SQL joins, CTEs, aggregate calculations, filtering

---

### 4. COVID-19 Dashboard (Tableau)

A Tableau packaged workbook visualizing global COVID trends using the same raw datasets used in the SQL analysis. The dashboard presents infection and death rates, vaccination progress, and comparisons across regions.

**File in repo:** `Covid-19 Dashboard.twbx`

**Tableau Link:** https://public.tableau.com/app/profile/towhidul.islam7095/viz/Covid-19Dashboard_17656241168560/Dashboard1 

**Skills used:** Tableau worksheets, parameters, filters, visual storytelling

---

### 5. Movie Correlation Analysis (Python)

An exploratory notebook that looks at what factors correlate with movie gross revenue.

**Skills used:** Python pandas, correlation, basic data visualization

---

### 6. Nashville Housing Data Cleaning (SQL + Excel)

A project where raw housing data is cleaned for future reporting. Steps include formatting date columns, standardizing categories, handling missing values, and removing duplicates.

**Skills used:** SQL cleaning techniques, Excel data preparation

---

### 7. Tableau Portfolio SQL Transformations

A set of SQL queries used to prepare datasets before uploading into a Tableau dashboard.
